pyquark
=========

## Installation

Run below command to install pyquark module:

    python setup.py install

```
+-----------------------------------------------------------------+
|                    example                                      |
|          +------------------------------------------------------+
|          | +----------------------------------------------------+

+----------+ +----------------------------------------------------+

|                         +------+                                |
|                         | Gpio |                                |
|                         +---+--+                                |
|                             ^                                   |
|                             | inherits                          |
|                     +-------+------+------------------+         |
|                     |              |                  |         |
|                     |              |                  |         |
|     +-------+  +----+----+   +-----+-----+   +--------+------+  |
|     |  Aio  |  |   Dio   |   |   PwmIO   |   |  MuxSelector  |  |

|         |         |                |                  ^         |
| depends +---------+                | depends          |         |
|         |                          |                  |         |
|         v                          v                  |         |
|    +----+------+   depends   +-----+-----+            |         |

```
